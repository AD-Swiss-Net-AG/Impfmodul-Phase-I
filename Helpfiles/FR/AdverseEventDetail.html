Effets secondaires

Vue détaillée d'un effet secondaire indésirable d'une vaccination enregistré dans le DEP d'un(e) patient(e).

Les données techniques d'un effet secondaire indésirable sont affichées dans la vue détaillée angezeigt. La vue détaillée sert de point d'entrée pour l'édition ou le commentaire, l'annulation ou le téléchargement d'une effet secondaire indésirbelpar la vaccination et permet les interactions suivantes:

  • Afficher la page de dialogue pour modifier les données de l'effet secondaire
  • Passez à la boîte de dialogue pour annuler les données des effet secondaire indésirable de vaccination
  • Téléchargement du jeu de données selon le format d'échange EPD en JSON.

La vue détaillée affiche les informations techniques suivantes:

  • Date: Date de l'apparition de l'effet secondaire indésirable
  • Maladie: La désignation de l'effet secondaire indésirable
  • Créé par: Nom de la personne qui a recueilli les renseignements de l'effet secondaire indésirable
  • Organisation: Nom de l'organisation où la vaccination a été effectuée.
