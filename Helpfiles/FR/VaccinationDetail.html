Vaccins

Vue détaillée d'une vaccination selectionnée du DEP d'un(e) patient(e).

Les données techniques d'une vaccination sélectionnée sont affichées dans la vue détaillée. La vue détaillée est le point de départ pour l'édition ou le commentaire, l'annulation ou le téléchargement d'une vaccination selectionnée et permet les actions suivantes:

  • Afficher la page de dialogue pour modifier les données de la vaccination sélectionnée
  • Affichage de la page de dialogue pour annuler des vaccinations erronées
  • Téléchargement du jeu de données selon le format d'échange EPD en JSON.

La vue détaillée affiche les informations techniques suivantes:

  • Date: Date de la vaccination
  • Vaccin: Le vaccin administré
  • Protection vaccinale: Une ou plusieurs maladies infectieuses contre lesquelles le vaccin est efficace
  • Numéro de lot: Le numéro de lot du vaccin
  • Dose: Numéro de la dose de vaccin administrée
  • Numéro de lot: Le numéro de lot du vaccin
  • Vacciné par: Nom et titre du professionnel de santé qui a administré le vaccin
  • Organisation: Nom de l'organisation où la vaccination a été effectuée.
