Facteurs de risque

Vue détaillée d'un facteur de risque sélectionnée à partir du DEP d'un(e) Patient(e).

Les données techniques d'un facteur de risque sélectionnée sont affichées dans la vue détaillée. La vue détaillée sert de point d'entrée pour l'édition ou le commentaire, l'annulation ou le téléchargement d'un facteur de risque et permet les interactions suivantes:

  • Passez à la boîte de dialogue pour modifier les données de le facteur de risque
  • Passez à la boîte de dialogue pour annuler le facteur de risque
  • Téléchargement du jeu de données selon le format d'échange EPD en JSON.

La vue détaillée affiche les informations techniques suivantes:

  • Date: Date de l'apparition de le facteur de risque par vaccination
  • Maladie: La désignation de le facteur de risque
  • Début: Moment de l'apparition des symptômes d'un facteur de risque
  • Fin: Date de la guérison
  • État clinique: L'état clinique, actif ou inactif
  • Créé par: Nom de la personne qui a recueilli les renseignements sur les facteurs de risque
  • Organisation: Nom de l'organisation où la vaccination a été effectuée.
